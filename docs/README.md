## 🚀 Quick Start

### For Developers

```bash
# Install dependencies
composer install
npm install

# Build assets
npm run build

# Development mode
npm run start
```

See [DEVELOPER-GUIDE.md](DEVELOPER-GUIDE.md) for complete setup.

### For Users

See [UPGRADE-GUIDE.md](UPGRADE-GUIDE.md) for installation and upgrade instructions.
